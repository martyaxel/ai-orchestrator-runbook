# AI Orchestrator Runbook

One file that sets up the **orchestrator + delegate** AI coding workflow:
**Claude Code** thinks, plans, delegates and verifies; **Codex (gpt-5.5)** executes —
basically free on subscription. Better output, no premium prices for mechanical work.

## How to use it (that's all)

1. Open your AI agent (Claude Code, Cursor — whatever you already have).
2. Hand it [runbook.md](runbook.md) — or just tell it:

   > Fetch https://raw.githubusercontent.com/martyaxel/ai-orchestrator-runbook/main/runbook.md and execute this runbook.

3. The agent does everything and only stops you for logins. ~3 minutes.

## What you need

- A Claude subscription (Pro/Max) + a ChatGPT subscription (Plus/Pro)
- Cursor or VS Code (free tier is fine — the intelligence comes from the subscriptions)

The runbook includes the `CLAUDE.md` model-routing table (cost / intelligence / taste),
so the orchestrator picks the right model for every task on its own.
